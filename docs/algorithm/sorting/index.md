# 📈 排序算法

排序是最基础也是最重要的算法之一，理解排序算法有助于掌握算法设计与分析的核心思想。

## 📚 内容规划

本章节将包含以下内容：

| 排序算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|----------|------------|------------|--------|
| 冒泡排序 | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(1) | 稳定 |
| 希尔排序 | O(n log n) | O(1) | 不稳定 |
| 快速排序 | O(n log n) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n) | 稳定 |
| 堆排序 | O(n log n) | O(1) | 不稳定 |
| 计数排序 | O(n + k) | O(k) | 稳定 |
| 基数排序 | O(nk) | O(n + k) | 稳定 |
