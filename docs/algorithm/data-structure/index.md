# 🏗️ 数据结构

数据结构是计算机存储、组织数据的方式，是算法实现的基础。

## 📚 内容规划

本章节将包含以下内容：

| 数据结构 | 说明 | 难度 |
|----------|------|------|
| 数组 | 连续内存存储，随机访问 O(1) | ⭐ |
| 链表 | 指针连接，插入删除 O(1) | ⭐ |
| 栈 | 后进先出 LIFO | ⭐ |
| 队列 | 先进先出 FIFO | ⭐ |
| 哈希表 | 键值对存储，平均 O(1) 查找 | ⭐⭐ |
| 树 | 层次结构，二叉树、BST、AVL | ⭐⭐⭐ |
| 堆 | 优先队列实现，堆排序 | ⭐⭐⭐ |
| 图 | 网络结构，DFS/BFS | ⭐⭐⭐⭐ |
